Ejemplos con for y while

IMPRIMIR NUMEROS IMPARES

Este programa en Java imprime todos los números impares desde 1 hasta 20 utilizando un bucle while. Comienza con i = 1 y, en cada iteración, muestra el valor de i en la consola. Luego incrementa i en 2 para saltar al siguiente número impar. El bucle finaliza cuando i supera 20.

IMPRIMIR NUMEROS

Este programa en Java imprime todos los números enteros desde 9 hasta 43 utilizando un bucle for. En cada iteración, incrementa el valor de i en 1 y lo muestra en la consola hasta alcanzar el límite superior.

CUENTA REGRESIVA

Este programa en Java realiza una cuenta regresiva desde un número ingresado por el usuario hasta llegar a cero. Utiliza un bucle while para imprimir cada número en orden descendente y reduce el valor en 1 en cada iteración. Incluye una pausa de 1 segundo (Thread.sleep(1000)) entre números para simular el tiempo real de la cuenta regresiva. Al finalizar, imprime el mensaje "¡Despegue!".

ADIVINA EL NUMERO

Este programa en Java es un juego para adivinar un número secreto generado aleatoriamente entre 1 y 100. Utiliza las clases Scanner y Random. El usuario ingresa intentos, y el programa da pistas indicando si el número secreto es mayor o menor que el intento. El ciclo do-while se repite hasta que el usuario adivine correctamente, momento en el que se muestra un mensaje de felicitación.

TABLA DE MULTIPLICAR

Este programa en Java genera la tabla de multiplicar de un número ingresado por el usuario. Utiliza un bucle while que recorre los números del 1 al 10, multiplicando el número ingresado por el valor del contador i en cada iteración. Luego, imprime el resultado en formato clásico de tabla (por ejemplo, 5 x 1 = 5).
